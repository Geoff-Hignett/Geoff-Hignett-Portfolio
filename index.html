My Work

Here is a selection of my recent work.

EXPERIENCE P2P

EXPERIENCE P2P is a multi-page website which provides reviews of peer-to-peer lending investment products. It was the first multi-page site I built using Bootstrap 4.

10 British Questions

10 British Questions is a random quiz app. It is rendered almost exclusively using CSS and JavaScript/jQuery with virtually no HTML. This was a great experience for practising pure CSS and DOM manipulation.

Riff World

Riff World is a guitar instruction and events website. It was made using the Materialize CSS framework and makes use of the Scrollfire plugin and Slider carousel feature.

Typing Trainer

Typing training is a game which tests the typing speed of the user. After a certain number of points scored the difficulty intensifies. It was made with JavaScipt and styled in Bootstrap 4.

Hignett's Rocking Horses

Hignett's Rocking Horses is a historical single-page website. It uses Bootstrap 4. It also makes use of the Scrollspy and SmoothScroll extensions.

Movie Quote Generator

This app generates random movie quotes. It makes AJAX calls to retrieve the quotes from an API on the Mashape market.
